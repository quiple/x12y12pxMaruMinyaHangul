# x12y12pxMaruMinyaHangul 「마루미냐 한글」

![MaruMinyaHangul](./doc/x12y12pxMaruMinyaHangul.png)

마루미냐 한글은 12픽셀 크기의 한국어&middot;일본어 비트맵 폰트입니다.

## 라이선스

모든 한글 및 2개의 대한민국 원 기호는 오픈 폰트 라이선스 1.1에 따라 이용할 수 있습니다.
